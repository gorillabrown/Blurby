## Title
Blurby — Speed Read Any Page

## Short Description (132 chars max)
Extract readable content from any web page and send it to Blurby for speed reading. Works locally or via cloud sync.

## Description
Blurby turns any web article into a speed-readable document. One click extracts the readable content from the current page and sends it straight to your Blurby desktop app.

**How it works:**
1. Visit any article, blog post, or web page
2. Click the Blurby icon or press Ctrl+Shift+B
3. The article appears in your Blurby library, ready to read

**Features:**
- Extracts clean, readable text from any web page using Mozilla's Readability engine
- Sends articles directly to your running Blurby desktop app via local connection
- Falls back to cloud sync (Google Drive or OneDrive) when the desktop app isn't running
- Page preview shows title, author, word count, and estimated reading time
- Right-click context menu: "Send to Blurby"
- Configurable keyboard shortcut (default: Ctrl+Shift+B)
- Choose reading mode: add to library, open in Focus mode, or open in Flow mode
- Recent sends history in the popup

**Privacy-first:**
- No data sent to any external servers — Blurby is a local-first app
- Content only extracted when you explicitly trigger it
- Cloud sync uses your own Google Drive or OneDrive account
- No analytics, no tracking, no telemetry

**Requires:** Blurby desktop app (Windows) for local connection, or a Google/Microsoft account for cloud sync.
