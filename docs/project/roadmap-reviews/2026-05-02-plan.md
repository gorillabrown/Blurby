### Stage 2: Feature Completion

#### SK-EINK-6A — E-Ink Foundation & Greyscale Runtime

*Already fully spec'd in ROADMAP.md (lines 1082–1145).* No changes needed — spec is dispatch-ready with tasks, edit-site coordinates, and SUCCESS CRITERIA.

- **What:** Decouple e-ink display behavior from theme system. New `einkMode` boolean setting, split CSS into behavioral (`[data-eink]`) and color (`[data-theme="eink"]`) layers, restructure ThemeSettings.
- **Why:** E-ink is currently locked to a single greyscale theme. Users with e-ink devices can't use dark mode + e-ink optimizations.
- **Prerequisites:** None (was blocked on TTS-7D, which completed long ago).
- **Done when:** 10 SUCCESS CRITERIA items per existing spec.
- **Effort:** M (~3). Full tier (tests + build).
- **Roster:** CLI/Zeus orchestration.
- **Source:** Existing ROADMAP.md spec.

#### SK-EINK-6B — E-Ink Reading Ergonomics & Mode Strategy

*Already fully spec'd in ROADMAP.md (lines 1148–1204).* Dispatch-ready.

- **What:** Stepped flow mode (chunk-based page advance), burst focus mode (phrase grouping), adaptive refresh heuristic.
- **Why:** All reading modes assume fast LCD. E-ink users get degraded experience in Flow and Focus.
- **Prerequisites:** EINK-6A.
- **Done when:** 10 SUCCESS CRITERIA items per existing spec.
- **Effort:** M (~3). Full tier.
- **Roster:** CLI/Zeus orchestration.
- **Source:** Existing ROADMAP.md spec.

#### SK-GOALS-6B — Reading Goal Tracking

*Already fully spec'd in ROADMAP.md (lines 1208–1270).* Dispatch-ready.

- **What:** Daily/weekly reading goal system with streak tracking, library progress widget, settings sub-page.
- **Why:** Users have no feedback loop for building reading habits. Queue tells what to read; goals tell how much.
- **Prerequisites:** None (independent of E-Ink).
- **Done when:** 10+ SUCCESS CRITERIA items per existing spec (item 11 was lost to binary corruption — reconstructable as "Widget displays correct progress percentage").
- **Effort:** M (~3). Full tier.
- **Roster:** CLI/Zeus orchestration.
- **Source:** Existing ROADMAP.md spec.

### Stage 3: Polish + Ship

#### SK-POLISH-1 — Desktop Small Wins Bundle
