# Evidence Index

This directory preserves curated canonical evidence copied out of `artifacts/`
before the bulk artifact tree is ignored. It is intentionally selective: small
decision records, summaries, ledgers, and trace JSONs are kept here; large WAVs,
Electron profiles, caches, package zips, and scratch outputs remain in
`artifacts/` and are not tracked.

## Curated Groups

- `tts-architecture-review/` - 2026-05-11 TTS architecture review synthesis,
  ledgers, appendices, final audit transcript, and remediation plan.
- `tts-eval-baseline/` - canonical TTS-EVAL baseline aggregate, gate, scenario
  summaries, and Qwen streaming baseline summaries.
- `moss-nano-13d-live-capture/` - live selected-Nano four-mode evidence and
  gate summaries supporting the 13e product decision.
- `moss-nano-6f-promotion-confirmation-v2/` - Nano 6F bounded soak promotion
  confirmation and lifecycle summary evidence.
- `moss-nano-12-live-four-mode-evidence/` - earlier four-mode summary retained
  as predecessor evidence to the 13d live capture.

Original source root: `artifacts/`.
