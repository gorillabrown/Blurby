# Kokoro Retirement Scorecard

This scorecard tracks the five gates that would need to be green before `KOKORO-RETIRE-2` could ever start.

**Program status:** Paused. The original retirement path was built around the current non-streaming local Qwen lane, but live validation on 2026-04-20 showed that lane is not a sufficient Kokoro replacement for sustained continuous CPU narration. Kokoro remains the explicit backup, and `KOKORO-RETIRE-1` / `KOKORO-RETIRE-2` should stay paused until the approved `Qwen3-TTS-streaming` successor lane exists and proves itself.
